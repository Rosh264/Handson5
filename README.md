# Handson 5 - Advanced VR Interactive Scene using Three.js

## 📌 Objective

To implement a VR-like interactive environment using Three.js with physics, user interaction, and 360° movement.

---

## 🧠 Description

This project simulates an advanced virtual 3D environment using Three.js.
It includes real-time rendering, keyboard-based movement, mouse-controlled 360° camera rotation, and basic physics like jumping and gravity.

The system provides a game-like experience similar to first-person (FPS) environments.

---

## 🚀 Features

* 3D environment with multiple objects
* Real-time rendering using WebGL
* 360° camera rotation using mouse (Pointer Lock API)
* First-person movement (W, A, S, D controls)
* Jump physics with gravity simulation
* Mouse interaction (click objects to change color)
* Dynamic object animation

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Three.js

---

## ▶️ How to Run

1. Open the project folder in VS Code
2. Right-click `index.html`
3. Click **Open with Live Server**
4. Click anywhere on the screen to enable mouse control

---

## 🎮 Controls

* Mouse Move → Look around (360° view)
* W → Move Forward
* S → Move Backward
* A → Move Left
* D → Move Right
* Space → Jump
* Mouse Click → Change object color
